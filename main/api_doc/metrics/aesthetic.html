imgutils.metrics.aesthetic¶

Overview:

A tool for measuring the aesthetic level of anime images, with the model obtained from skytnt/anime-aesthetic.

This is an overall benchmark of all the operations in aesthetic models:

get_aesthetic_score¶

imgutils.metrics.aesthetic.get_aesthetic_score(image: Union[str, os.PathLike, bytes, bytearray, BinaryIO, PIL.Image.Image])[source]¶
Overview:

Get aesthetic score for image.

Parameters:

image – Original image.

Returns:

Score of aesthetic.

Examples::
>>> from imgutils.metrics import get_aesthetic_score
>>>
