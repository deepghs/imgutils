imgutils.tagging.wd14¶

Overview:

Tagging utils based on wd14 v2, inspired by SmilingWolf/wd-v1-4-tags .

get_wd14_tags¶

imgutils.tagging.wd14.get_wd14_tags(image: Union[str, os.PathLike, bytes, bytearray, BinaryIO, PIL.Image.Image], model_name: str = 'ConvNextV2', general_threshold: float = 0.35, character_threshold: float = 0.85)[source]¶
Overview:

Tagging image by wd14 v2 model. Similar to SmilingWolf/wd-v1-4-tags .

Parameters:

  • image – Image to tagging.

  • model_name – Name of the mode, should be one of the SwinV2, ConvNext, ConvNextV2 or ViT, default is ConvNextV2.

  • general_threshold – Threshold for default tags, default is 0.35.

  • character_threshold – Threshold for character tags, default is 0.85.

Returns:

Tagging results for levels, features and characters.
