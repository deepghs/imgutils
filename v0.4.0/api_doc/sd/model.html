imgutils.sd.model

Overview:

A utility for reading and writing metadata from/to model files in the A41 WebUI format.

Note

torch and safetensors are required by this model. Please install them with the following command before start using this module.

pip install dghs-imgutils[model]

save_with_metadata

imgutils.sd.model.save_with_metadata(src_model_file: str, dst_model_file: str, metadata: Dict[str, str], clear: bool = False)[source]

Saves a model file with metadata. Optionally, existing metadata can be cleared before adding new metadata.

Parameters:

  • src_model_file (str) – The path to the source model file.

  • dst_model_file (str) – The path to save the new model file.

  • metadata (Dict[str, str]) – The metadata to add to the model file.

  • clear (bool) – Whether to clear existing metadata before adding new metadata. Default is False.
