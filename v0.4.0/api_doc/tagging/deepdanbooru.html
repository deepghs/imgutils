imgutils.tagging.deepdanbooru

Overview:

Tagging utils based on deepdanbooru.

Warning

Due to the usage of an outdated model and training data in deepdanbooru, its performance is limited, and it is not suitable for use as the main tagging model anymore. The integration of this model within the present project serves only as a baseline for comparison, and it is advisable to avoid using this model extensively in practical applications.

get_deepdanbooru_tags

imgutils.tagging.deepdanbooru.get_deepdanbooru_tags(image: str | PathLike | bytes | bytearray | BinaryIO | Image, use_real_name: bool = False, general_threshold: float = 0.5, character_threshold: float = 0.5, drop_overlap: bool = False)[source]
Overview:

Get tags for anime image based on deepdanbooru model.

Parameters:

  • image – Image to tagging.

  • use_real_name – Use real name on danbooru. Due to the renaming and redirection of many tags on the Danbooru website after the training of deepdanbooru, it may be necessary to use the latest tag names in some application scenarios. The default value of False indicates the use of the original tag names.

  • general_threshold – Threshold for default tags, default is 0.35.

  • character_threshold – Threshold for character tags, default is 0.85.

  • drop_overlap – Drop overlap tags or not, default is False.

Returns:

Tagging results for levels, features and characters.
