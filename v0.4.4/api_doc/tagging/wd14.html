imgutils.tagging.wd14

Overview:

Tagging utils based on wd14 v2, inspired by SmilingWolf/wd-v1-4-tags .

get_wd14_tags

imgutils.tagging.wd14.get_wd14_tags(image: str | PathLike | bytes | bytearray | BinaryIO | Image, model_name: str = 'SwinV2_v3', general_threshold: float = 0.35, general_mcut_enabled: bool = False, character_threshold: float = 0.85, character_mcut_enabled: bool = False, no_underline: bool = False, drop_overlap: bool = False)[source]
Overview:

Get tags for an image with wd14 taggers. Similar to SmilingWolf/wd-v1-4-tags .

Parameters:

  • image (ImageTyping) – The input image.

  • model_name (str) – The name of the model to use.

  • general_threshold (float) – The threshold for general tags.

  • general_mcut_enabled (bool) – If True, applies MCut thresholding to general tags.

  • character_threshold (float) – The threshold for character tags.

  • character_mcut_enabled (bool) – If True, applies MCut thresholding to character tags.

  • no_underline (bool) – If True, replaces underscores in tag names with spaces.

  • drop_overlap (bool) – If True, drops overlapping tags.

Returns:

A tuple containing dictionaries for rating, general, and character tags with their probabilities.

Return type:

Tuple[Dict[str, float], Dict[str, float], Dict[str, float]]
