imgutils.tagging.mldanbooru¶

Overview:

Tagging utils based on ML-danbooru which is provided by 7eu7d. The code is here: 7eu7d7/ML-Danbooru .

get_mldanbooru_tags¶

imgutils.tagging.mldanbooru.get_mldanbooru_tags(image: Union[str, os.PathLike, bytes, bytearray, BinaryIO, PIL.Image.Image], use_real_name: bool = False, threshold: float = 0.7, size: int = 448, keep_ratio: bool = False)[source]¶
Overview:

Tagging image with ML-Danbooru, similar to deepghs/ml-danbooru-demo.

Parameters:

  • image – Image to tagging.

  • use_real_name – Use real name on danbooru. Due to the renaming and redirection of many tags on the Danbooru website after the training of deepdanbooru, it may be necessary to use the latest tag names in some application scenarios. The default value of False indicates the use of the original tag names.

  • threshold – Threshold for tags, default is 0.7.

  • size – Size when passing the resized image into model, default is 448.

  • keep_ratio – Keep the original ratio between height and width when passing the image into model, default is False.
