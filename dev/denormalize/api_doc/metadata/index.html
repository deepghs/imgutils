imgutils.metadataÔÉÅ

This module provides functionality for reading and writing generation information (geninfo) and metadata using various methods.

It includes functions for handling generation information in different formats (parameters, EXIF, GIF) and working with LSB (Least Significant Bit) metadata.

The module is designed to be used in image processing and generation tasks, particularly in the context of AI-generated images.
