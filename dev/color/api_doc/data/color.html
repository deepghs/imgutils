imgutils.data.color

extract_main_colors

imgutils.data.color.extract_main_colors(image: str | PathLike | bytes | bytearray | BinaryIO | Image, n: int = 12, scale: int = 400, mode: str = 'LAB', force_background: str = 'white', fmt=('image', 'colors'), **kwargs)[source]

Extracts the main colors from the input image using KMeans clustering.

Parameters:

  • image (ImageTyping) – The input image.

  • n (int) – The number of main colors to extract. (default: 12)

  • scale (int) – The scale factor for resizing the image. (default: 400)

  • mode (str) – The mode of the input image. (default: ‘LAB’)

  • force_background (str) – The background color to force. (default: ‘white’)

  • fmt (tuple) – The format of the output. (default: (‘image’, ‘colors’))

  • kwargs – Additional keyword arguments for image loading and resizing.

Returns:

The extracted main colors and the resized image.

Return type:

tuple
